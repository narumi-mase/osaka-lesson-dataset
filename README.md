# Osaka Lesson Dataset

This repository provides a structured dataset of junior high school lesson plans published by Osaka City Education Center.

## Purpose

The dataset was constructed for research on educational AI agents and lesson design support.  
It aims to make lesson plans easier to search, compare, and reuse by structuring lesson procedures and teaching material metadata.
